
 ViralShield UI

ViralShield UI is a cybersecurity-themed browser extension interface that displays real-time phishing and scam threat analysis.

It connects to a backend AI engine and visualizes structured threat intelligence in a focused, high-contrast security console layout.


 What It Does

* Sends selected webpage text to backend
* Receives structured threat analysis
* Displays:

  * Threat Index (0–100 risk score)
  * Classification (Safe / Suspicious / Dangerous)
  * Detected red flags
  * Reason for classification
  * Suggested countermeasures



Design Style

* Dark cybersecurity console theme
* Minimal, single-task focused interface
* Extension side-panel optimized
* No SaaS dashboard layout
* Clean, high-intensity signal display




* React + TypeScript
* Vite
* Chrome Extension (Manifest V3)
* Custom CSS (cybersecurity theme)

---

ViralShield UI acts as the visual defense layer — translating AI threat intelligence into immediate, actionable insight.
